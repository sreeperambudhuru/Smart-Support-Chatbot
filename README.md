# Smart Support Assistant 🤖

A professional Customer Support Chatbot prototype built with **Python** and **Streamlit**. 

## ✨ Features
* **Intent Recognition:** Identifies user goals like "Track Order" or "Refund Request."
* **Entity Extraction:** Uses Regex to automatically capture 5-digit Order IDs from sentences.
* **Smart Fallback:** Provides a graceful response when a query isn't recognized.

## 🛠️ Tech Stack
* **Frontend:** Streamlit (Web UI)
* **Backend:** Python
* **NLP Logic:** Keyword Mapping & Regular Expressions (Regex)

## 🚀 How to Run
1. Install requirements:  
   `pip install -r requirements.txt`
2. Launch the app:  
   `streamlit run app.py`
